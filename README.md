# 🧠 Migraine Prediction ML

A machine learning-based migraine prediction system built using real-world survey data and a complete preprocessing pipeline.

---

## 📌 Overview

This project focuses on predicting the likelihood of migraines based on user lifestyle, health patterns, and triggers collected through a structured survey.

The system applies data preprocessing techniques and machine learning algorithms to analyze patterns and generate predictions.

---

## 📊 Dataset

- File: `migraine prediction survey.xlsx`
- Total Responses: **100**
- Data Type: Survey-based (user input)

### Features include:
- Sleep patterns
- Stress levels
- Screen time
- Water intake
- Migraine history

---

## ⚙️ Workflow

1. **Data Collection**
   - Survey responses collected and stored in Excel format

2. **Data Preprocessing**
   - Handling missing values
   - Encoding categorical variables
   - Feature selection

3. **Model Building**
   - Applied machine learning algorithms (e.g., Logistic Regression / Decision Tree / etc.)

4. **Evaluation**
   - Accuracy and performance metrics used to evaluate the model
